# Start Your Cyber Security Journey

## Overview

Introduction to fundamental cybersecurity concepts through interactive exercises covering offensive security, defensive security and information searching techniques.

This module provides an initial understanding of cybersecurity roles, security workflows and the importance of both attacking and defending systems.

---

## Offensive Security Intro

### Overview

Introduction to ethical hacking concepts through a safe TryHackMe laboratory environment.

### Concepts Learned

- Difference between offensive and defensive security.
- Ethical hacking methodology.
- Basic web application security concepts.

### Tools and Techniques

- Browser-based security testing.
- Identifying hidden pages.
- Understanding attack surfaces.

### Key Takeaways

This room introduced the role of a penetration tester and the importance of performing security assessments legally and responsibly.

---

## Defensive Security Intro

### Overview

Introduction to defensive cybersecurity concepts through a simulated security incident investigation in the TryHackMe environment.

### Concepts Learned

- Role of defensive security teams.
- Basic incident investigation concepts.
- Identifying and analysing security events.
- Difference between offensive and defensive security.

### Skills Developed

- Security monitoring mindset.
- Basic threat analysis.
- Understanding defensive security workflows.

### Key Takeaways

This room provided an introduction to the role of cybersecurity defenders and how security teams investigate and respond to potential threats.

---

## Search Skills

### Overview

Introduction to efficient information searching techniques and the use of specialised search engines and technical documentation.

### Concepts Learned

- Effective information searching.
- Using search engines for technical research.
- Finding relevant cybersecurity documentation.

### Skills Developed

- Research skills.
- Technical information gathering.
- Documentation lookup.

### Key Takeaways

Developing effective search skills is essential in cybersecurity for finding documentation, investigating incidents and learning new technologies.
