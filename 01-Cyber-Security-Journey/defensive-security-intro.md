# Defensive Security Intro

## Overview
Introduction to defensive cybersecurity concepts through a simulated security incident investigation in the TryHackMe environment.

## Concepts learned
- Role of defensive security teams.
- Basic incident investigation concepts.
- Identifying and analysing security events.
- Difference between offensive and defensive security.

## Skills developed
- Security monitoring mindset.
- Basic threat analysis.
- Understanding defensive security workflows.

## Key takeaways
This room provided an introduction to the role of cybersecurity defenders and how security teams investigate and respond to potential threats.
