# Offensive Security Intro

## Overview
Introduction to ethical hacking concepts through a safe TryHackMe laboratory environment.

## Concepts learned
- Difference between offensive and defensive security.
- Ethical hacking methodology.
- Basic web application security concepts.

## Tools and techniques
- Browser-based security testing.
- Identifying hidden pages.
- Understanding attack surfaces.

## Key takeaways
This room introduced the role of a penetration tester and the importance of performing security assessments legally and responsibly.
